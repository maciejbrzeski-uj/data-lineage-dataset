# data-lineage-dataset
Published data for a publication sent to the VLDB conference.

The database comes from an public example, Adventure Works [adventure-works](https://github.com/microsoft/sql-server-samples/tree/master/samples/databases/adventure-works), made by Microsoft.

The database was enriched with procedures to display data lineage. Next, the data was processed to obtain files containing lineage links.

The owner of the examples remains Informatica.
